# Mister Chaos GPS Sender v1.2 Map Toggle

Diese Version kann zusätzlich den Kartenmodus der Webseite steuern.

Neue Buttons:
- GPS AN
- GPS AUS
- Kartenmodus AN
- Kartenmodus AUS
- Letzten bekannten Standort senden

Wichtig:
- GPS AUS beendet die Standortübertragung.
- Kartenmodus AUS versteckt nur die Karte auf der Webseite.
- GPS kann dabei weiterlaufen.
- Kartenmodus AN zeigt die Karte wieder auf der Webseite.

WordPress braucht dazu Theme v57 oder neuer.
